# Block-Breaker

The structure of the game states of my project:

 1. **__Start Scene__**
 2. **__Game levels__**
  - from level 1 to level 5
 3. **__Game Over__**
 
 The scripts i used :
 
 1. Ball
 2. Block
 3. GameSession
 4. Padlle
 5. Level
 6. LoseCollider
 7. SceneLoader
 
 The fonts I used:
 
 1. "retro SDF"
 2. "RetroCaps"
